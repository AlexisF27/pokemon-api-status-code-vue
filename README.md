# User Stories for API pokemon status code

1. [x] Install vue.js
2. [ ] Add html base project



- Clone this project
- Then run:
  
```shell
$ npm init
```


- Replace "scripts" attribute so you can start nodemon with dev option

```package.json
"scripts": {
    "dev": "nodemon server",
    "start": "node server"
  },
```
